# 튜터리얼 3: 투자수익률 계산 및 시뮬레이션

## 개요

- **목적:** 부동산 투자의 실질 수익률을 정확히 계산하고, 다양한 시나리오별 시뮬레이션을 수행합니다.
- **대상 독자:** 부동산 투자 수익률에 관심 있는 투자자, 갭투자 및 대출 레버리지를 활용한 투자 전략을 검토하는 분
- **소요 시간:** 약 50-60분
- **난이도:** 중급
- **사용 플러그인:**
  - `mcp-kr-realestate` - 실거래가 조회 및 시세 데이터 수집
  - `data:analyze` - 수익률 데이터 분석 및 테이블 생성
  - `finance:variance-analysis` - 예상 vs 실제 수익률 차이 분석

---

## 사전 준비

### 플러그인 설정 확인

`mcp-kr-realestate` 플러그인이 정상적으로 설정되어 있어야 합니다. 아직 설정하지 않았다면 [튜터리얼 1](./튜터리얼1_아파트_실거래가_조회_및_시세분석.md)을 먼저 완료해 주세요.

### 필요한 사전 지식

- 매매가, 전세가, 월세의 개념
- 기본적인 부동산 세금 구조 (취득세, 양도소득세)
- 대출 금리 및 원리금 상환 개념

---

## 핵심 개념

### 부동산 투자 수익 구조

부동산 투자 수익은 크게 세 가지로 구성됩니다.

| 수익 유형 | 설명 | 예시 |
|-----------|------|------|
| **시세차익** | 매수가 대비 매도가 상승분 | 28억에 매수 → 32억에 매도 = 4억 차익 |
| **임대 수익** | 전세 또는 월세를 통한 수익 | 월세 150만원 × 12개월 = 연 1,800만원 |
| **레버리지 효과** | 타인 자본(전세금, 대출)을 활용한 수익 극대화 | 자기자본 13억으로 28억 자산 운용 |

### 비용 구조

투자 수익률을 정확히 계산하려면 모든 비용을 반영해야 합니다.

#### 취득세 (2024-2025년 기준)

| 주택 수 | 조정대상지역 | 비조정대상지역 | 비고 |
|---------|-------------|---------------|------|
| 1주택 | 1-3% | 1-3% | 6억 이하 1%, 6-9억 1-3%, 9억 초과 3% |
| 2주택 | 8% | 1-3% | 조정대상지역 중과 |
| 3주택 이상 | 12% | 8% | 중과세율 적용 |
| 법인 | 12% | 12% | 주택 수 무관 |

> **참고:** 취득세에는 지방교육세(0.1-0.4%)와 농어촌특별세(0.2%)가 추가됩니다. 실질 세율은 위 표보다 약 0.3-0.6%p 높습니다.

#### 보유세

- **재산세:** 과세표준 × 0.1-0.4% (주택 공시가격 기준, 6월 1일 기준 부과)
- **종합부동산세:** 공시가격 합산 12억 초과 시 0.5-2.7% (1주택자 기본공제 12억, 다주택자 9억)
- **종합부동산세 세율 (2024년):**

| 과세표준 구간 | 일반세율 (2주택 이하) | 중과세율 (3주택 이상) |
|--------------|---------------------|---------------------|
| 3억 이하 | 0.5% | 0.5% |
| 3-6억 | 0.7% | 0.7% |
| 6-12억 | 1.0% | 1.0% |
| 12-25억 | 1.3% | 2.0% |
| 25-50억 | 1.5% | 3.0% |
| 50-94억 | 2.0% | 4.0% |
| 94억 초과 | 2.7% | 5.0% |

#### 양도소득세

| 보유기간 | 기본세율 | 비고 |
|---------|---------|------|
| 1년 미만 | 45% | 단기 매매 중과 |
| 1-2년 | 기본세율(6-45%) | - |
| 2년 이상 | 기본세율(6-45%) | 장기보유특별공제 적용 가능 |

**양도소득세 기본세율 (2024년):**

| 과세표준 | 세율 | 누진공제 |
|---------|------|---------|
| 1,400만원 이하 | 6% | - |
| 1,400-5,000만원 | 15% | 126만원 |
| 5,000만원-8,800만원 | 24% | 576만원 |
| 8,800만원-1.5억 | 35% | 1,544만원 |
| 1.5억-3억 | 38% | 1,994만원 |
| 3억-5억 | 40% | 2,594만원 |
| 5억-10억 | 42% | 3,594만원 |
| 10억 초과 | 45% | 6,594만원 |

> **1세대 1주택 비과세:** 2년 이상 보유(조정대상지역은 2년 거주 요건 추가) 시 양도차익 12억원까지 비과세. 12억 초과분에 대해서만 과세.

#### 기타 비용

- **중개수수료:** 매매가 × 0.4-0.7% (거래가액에 따라 상한 요율 상이)
- **대출 이자:** 연 4.0-5.5% (2024-2025년 주택담보대출 평균 금리)
- **수선유지비:** 연간 매매가의 약 0.3-0.5%

## Step 1: 기본 투자수익률 계산

먼저 대상 물건의 실거래가 데이터를 조회하고, 기본적인 투자수익률을 계산합니다.

### 실거래가 데이터 조회

Claude Code에서 다음 프롬프트를 입력합니다:

```text
강남구 대치동 은마아파트 76㎡의 최근 3년간 실거래가를 조회해줘.
그리고 현재 시세와 전세가도 함께 확인해줘.
```

`mcp-kr-realestate` 플러그인이 실거래가 데이터를 조회하여 다음과 같은 결과를 반환합니다:

```text

| 거래일     | 층  | 매매가(만원)  | 전세가(만원)  |
|-----------|-----|-------------|-------------|
| 2025.12   | 15  | 280,000     | 150,000     |
| 2025.09   | 8   | 275,000     | 148,000     |
| 2025.06   | 22  | 272,000     | 145,000     |
| 2025.01   | 11  | 265,000     | 140,000     |
| 2024.08   | 18  | 258,000     | 138,000     |
| 2024.03   | 5   | 250,000     | 135,000     |
| 2023.10   | 14  | 245,000     | 132,000     |
| 2023.05   | 9   | 240,000     | 130,000     |

■ 현재 추정 시세: 약 28억원
■ 현재 전세 시세: 약 15억원
■ 전세가율: 53.6%
```

### 투자수익률 계산 프롬프트

```text
강남구 대치동 은마아파트 76㎡ 투자 수익률을 계산해줘.
조건은 다음과 같아:
- 매매가: 28억원
- 전세가: 15억원
- 보유기간: 3년
- 예상 매도가: 32억원
- 1주택자, 2년 실거주 예정
- 취득 시 중개수수료, 매도 시 중개수수료 포함
모든 세금과 비용을 반영한 세전/세후 수익률을 모두 보여줘.
```

## Step 2: 갭투자 수익률 분석

갭투자란 매매가와 전세가의 차이(갭)만큼만 자기 자본을 투입하고, 전세금을 레버리지로 활용하는 투자 방식입니다.

### 갭투자 구조

```text
┌──────────────────────────────────┐
│         매매가 28억원             │
│                                  │
│  ┌──────────────────────────┐   │
│  │    전세금 15억원 (53.6%)  │   │ ← 세입자 자금 (레버리지)
│  │    = 타인 자본            │   │
│  └──────────────────────────┘   │
│  ┌──────────────────────────┐   │
│  │    갭 13억원 (46.4%)      │   │ ← 자기 자본 (실제 투자금)
│  │    = 투자 원금            │   │
│  └──────────────────────────┘   │
└──────────────────────────────────┘
```

### 갭투자 수익률 계산 프롬프트

```text
갭투자 관점에서 수익률을 분석해줘.
조건:
- 매매가: 28억원
- 전세가: 15억원 (전세가율 53.6%)
- 자기자본: 13억원 (갭)
- 보유기간: 3년
- 예상 매도가: 32억원
- 1주택자, 2년 실거주
- 전세 레버리지 15억원 활용
전세가율별 수익률 비교도 함께 보여줘.
```

### 전세가율별 수익률 비교

전세가율이 높을수록 갭이 작아지고, 레버리지 효과가 극대화됩니다.

```text

| 50%     | 14.0억  | 14.0억     | 1.89억    | 13.49%    | 4.32%       |
| 53.6%   | 15.0억  | 13.0억     | 1.89억    | 14.52%    | 4.62%       |
| 60%     | 16.8억  | 11.2억     | 1.89억    | 16.86%    | 5.32%       |
| 70%     | 19.6억  | 8.4억      | 1.89억    | 22.48%    | 6.98%       |
| 80%     | 22.4억  | 5.6억      | 1.89억    | 33.72%    | 10.17%      |
```

> **주의:** 전세가율이 높을수록 수익률은 높아지지만, 전세가 하락 시 역전세(전세금 반환 불능) 리스크도 커집니다. 전세가율 70% 이상은 고위험 구간으로 분류됩니다.

## Step 3: 대출 레버리지 분석

갭투자에서 자기자본(갭) 부분에 주택담보대출을 활용하면 실질 투자금을 더 줄일 수 있습니다.

### 대출 레버리지 프롬프트

```text
주택담보대출 활용 시 레버리지 효과를 분석해줘.
조건:
- 매매가: 28억원
- 전세가: 15억원
```

```text
 전세가 15억 → 12억 하락 시:
 - 전세금 반환을 위해 추가 자금 3억 필요
 - 자기자본 13억 + 추가 3억 = 16억 투자
 - 갭투자 수익률 급감

 전세가 15억 → 10억 하락 시:
 - 추가 자금 5억 필요 (깡통전세 위험)
 - 자금 동원 불가 시 법적 분쟁 발생 가능
```

## 수식 정리

투자수익률 계산에 사용되는 주요 수식을 정리합니다.

| 항목 | 수식 | 설명 |
|------|------|------|
| **단순 수익률** | `(매도가 - 매수가 - 총비용) / 투자원금 × 100` | 보유기간 전체 수익률 |
| **연환산 수익률 (CAGR)** | `((1 + 총수익률)^(1/보유년수) - 1) × 100` | 복리 기준 연간 수익률 |
| **갭투자 수익률** | `세후 순수익 / (매매가 - 전세가) × 100` | 자기자본(갭) 대비 수익률 |
| **Cap Rate** | `연간 순영업이익(NOI) / 매매가 × 100` | 매매가 대비 임대 수익률 |
| **레버리지 수익률** | `(시세차익 - 이자비용 - 세금) / 자기자본 × 100` | 대출 활용 시 수익률 |
| **월 원리금균등상환액** | `대출원금 × r × (1+r)^n / ((1+r)^n - 1)` | r=월이율, n=총상환개월수 |
| **전세가율** | `전세가 / 매매가 × 100` | 매매가 대비 전세가 비율 |
| **레버리지 배수** | `총 자산가치 / 자기자본` | 자기자본 대비 운용 자산 비율 |
| **세후 실질 수익률** | `(총수익 - 총비용 - 총세금) / 자기자본 × 100` | 모든 비용/세금 반영 |
| **손익분기 상승률** | `((매수가 + 총비용 + 세금) / 매수가)^(1/n) - 1` | 원금 보전을 위한 최소 상승률 |

### 양도소득세 계산 공식

```text
과세 대상 양도차익 = 양도차익 × (양도가액 - 12억) / 양도가액   (1주택 비과세 적용 시)
장기보유특별공제 = 과세 대상 양도차익 × (보유공제율 + 거주공제율)
과세표준 = 과세 대상 양도차익 - 장기보유특별공제 - 기본공제(250만원)
양도소득세 = 과세표준 × 세율 - 누진공제
총 세금 = 양도소득세 × 1.1 (지방소득세 10% 포함)
```

1. **세율 변동:** 부동산 세제는 정부 정책에 따라 수시로 변경됩니다. 본 튜터리얼의 세율은 2024-2025년 기준이며, 실제 투자 시에는 반드시 최신 세법을 확인하세요.
2. **시뮬레이션 한계:** 시세 변동률은 과거 데이터 기반 가정이며, 미래 시세를 보장하지 않습니다.
3. **개인 상황:** 주택 수 판정, 조정대상지역 지정, 비과세 요건 등은 개인 상황에 따라 달라집니다. 반드시 세무사와 상담하세요.
4. **대출 규제:** LTV, DTI, DSR 등 대출 규제는 시기와 지역에 따라 다릅니다. 실제 대출 가능 금액은 은행 상담을 통해 확인하세요.
