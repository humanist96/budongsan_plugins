# 튜터리얼 1: 아파트 실거래가 조회 및 시세 분석

## 개요

- **목적:** Claude Code와 mcp-kr-realestate MCP 서버를 활용하여 한국 아파트 실거래가를 조회하고 시세 추세를 분석하는 방법을 학습합니다.
- **대상 독자:** 부동산 투자에 관심이 있는 개발자/데이터 분석가
- **소요 시간:** 약 30-40분
- **난이도:** 초급
- **사용 플러그인:** [mcp-kr-realestate](https://github.com/ChangooLee/mcp-kr-realestate) - 한국 부동산 투자 분석 MCP 서버

### 이 튜터리얼에서 배우는 것

1. 국토교통부 실거래가 API와 한국은행 경제지표 API 키를 발급받고 MCP 서버를 설정하는 방법
2. Claude Code에서 자연어 프롬프트로 특정 아파트 단지의 실거래가를 조회하는 방법
3. 조회된 데이터를 기반으로 시계열 추세 분석 및 시세 변동률을 해석하는 방법
4. 주변 단지와의 비교 분석을 통해 투자 인사이트를 도출하는 방법

---

## 사전 준비

### 1. API 키 발급

이 튜터리얼에서는 두 가지 공공 데이터 API를 사용합니다. 모두 무료로 발급받을 수 있습니다.

#### 1-1. 국토교통부 실거래가 API 키 발급

국토교통부 실거래가 API는 공공데이터포털(data.go.kr)을 통해 제공됩니다.

**발급 절차:**

1. [공공데이터포털](https://www.data.go.kr)에 접속하여 회원가입 및 로그인합니다.
2. 상단 검색창에 "국토교통부 아파트매매 실거래 상세 자료"를 검색합니다.
3. 검색 결과에서 해당 API를 클릭한 후, **[활용신청]** 버튼을 클릭합니다.
4. 활용 목적을 "학습/연구" 등으로 기재하고 신청을 완료합니다.
5. 신청 후 **마이페이지 > 오픈API > 인증키 발급현황**에서 발급된 인코딩/디코딩 키를 확인합니다.
6. **디코딩 키(Decoding)**를 복사하여 보관합니다.

> **참고:** 활용 신청 승인은 보통 즉시 또는 1-2시간 내에 완료됩니다. 일일 트래픽 제한은 기본 1,000건이며, 추가 신청 시 확장 가능합니다.

추가로 다음 API도 함께 신청해 두면 전세가 분석 등에 유용합니다:

- 국토교통부 아파트 전월세 실거래 상세 자료
- 국토교통부 아파트매매 실거래자료

#### 1-2. 한국은행 API 키 발급

한국은행 경제통계시스템(ECOS) API는 기준금리, 소비자물가지수 등 거시경제 지표 분석에 활용됩니다.

**발급 절차:**

1. [한국은행 경제통계시스템](https://ecos.bok.or.kr)에 접속합니다.
2. 상단 메뉴에서 **[개발자센터]**를 클릭합니다.
3. **[인증키 신청]** 페이지에서 이메일 주소와 사용 목적을 입력합니다.
4. 입력한 이메일로 인증키가 발송됩니다.
5. 이메일에서 인증키를 확인하여 보관합니다.

> **참고:** 한국은행 API는 일일 호출 제한이 100,000건으로 넉넉합니다.

### 2. MCP 서버 설치 및 설정

#### 2-1. mcp-kr-realestate 설치

pip를 사용하여 설치하는 방법:

```bash
pip install mcp-kr-realestate
```

또는 GitHub에서 직접 클론하여 설치하는 방법:

```bash
git clone https://github.com/ChangooLee/mcp-kr-realestate.git
cd mcp-kr-realestate
pip install -r requirements.txt
```

#### 2-2. 환경변수 설정

프로젝트 루트 디렉터리에 `.env` 파일을 생성하거나, 셸 환경변수로 API 키를 등록합니다.

**방법 1: .env 파일 생성**

```bash
# .env
MOLIT_API_KEY=your_molit_api_key_here
BOK_API_KEY=your_bok_api_key_here
```

**방법 2: 셸 환경변수 설정 (zsh 기준)**

```bash
# ~/.zshrc에 추가
export MOLIT_API_KEY="your_molit_api_key_here"
export BOK_API_KEY="your_bok_api_key_here"
```

설정 후 셸을 재시작하거나 `source ~/.zshrc`를 실행합니다.

#### 2-3. Claude Code MCP 서버 설정

`.claude/settings.json` 파일에 MCP 서버 정보를 등록합니다. 이 설정을 통해 Claude Code가 mcp-kr-realestate 서버와 통신할 수 있게 됩니다.

```json
{
  "mcpServers": {
    "kr-realestate": {
      "command": "python",
      "args": ["mcp-servers/kr-realestate/main.py"],
      "env": {
        "MOLIT_API_KEY": "${MOLIT_API_KEY}",
        "BOK_API_KEY": "${BOK_API_KEY}"
      }
    }
  }
}
```

> **주의:** `args`의 경로는 실제 설치된 경로에 맞게 수정해야 합니다. pip로 설치한 경우와 git clone으로 설치한 경우 경로가 다를 수 있습니다.

#### 2-4. 설치 확인

Claude Code를 실행하여 MCP 서버가 정상적으로 연결되었는지 확인합니다.

```
> claude

# Claude Code 프롬프트에서 다음을 입력합니다:
mcp-kr-realestate 서버가 정상적으로 연결되어 있는지 확인해줘
```

정상적으로 연결된 경우, Claude가 사용 가능한 도구 목록을 보여줍니다. 주요 도구:

- `get_apartment_trades` - 아파트 매매 실거래가 조회
- `get_apartment_rent` - 아파트 전월세 실거래가 조회
- `get_region_code` - 지역(법정동) 코드 조회
- `get_economic_indicator` - 한국은행 경제지표 조회

## Step 1: 지역 코드 이해하기

### 법정동 코드 체계

국토교통부 실거래가 API는 법정동 코드를 기반으로 지역을 식별합니다. 코드는 10자리 숫자로 구성되며, 다음과 같은 계층 구조를 가집니다:

| 구분 | 자릿수 | 예시 | 설명 |
|------|--------|------|------|
| 시도 | 2자리 | 11 | 서울특별시 |
| 시군구 | 3자리 | 680 | 강남구 |
| 읍면동 | 5자리 | 10300 | 대치동 |

따라서 "서울특별시 강남구 대치동"의 전체 법정동 코드는 `1168010300`이 됩니다.

## Step 3: 시계열 추세 분석

**Claude Code 프롬프트:**

```

3. 추세 분석 요약

- 전반적 상승 추세: 2025년 3월 대비 2026년 1월 기준 약 15.6% 상승했습니다.
- 상승 가속 구간: 2025년 9~11월에 상승폭이 두드러졌으며,
  이는 정부의 대출 규제 완화 발표와 시기적으로 일치합니다.
- 일시적 조정: 2025년 12월에 소폭 하락(-4.29%)이 있었으나,
  연말 거래 감소에 따른 일시적 현상으로 판단됩니다.
- 이동평균선이 우상향을 유지하고 있어 중기적 상승 추세가 지속되고 있습니다.
```

### 분석 방법 설명

| 분석 기법 | 설명 | 활용 |
|-----------|------|------|
| 이동평균 (Moving Average) | 최근 N개월 가격의 평균으로 단기 변동을 완화하여 추세를 파악 | 3개월 이동평균으로 상승/하락 추세 확인 |
| 전월 대비 변동률 | (당월 평균가 - 전월 평균가) / 전월 평균가 x 100 | 월별 가격 변화 속도 파악 |
| 중위가격 | 전체 거래를 가격순으로 정렬했을 때 중간값 | 극단값에 영향을 덜 받는 대표가격 |
| 표준편차 | 가격의 산포도 (변동성 지표) | 가격 안정성 판단 기준 |

### 변동률 해석 가이드

분석 결과를 해석할 때 다음 사항을 고려합니다:

**전월 대비 변동률 해석 기준:**

| 변동률 범위 | 시장 상태 | 투자 시사점 |
|------------|----------|------------|
| +3% 이상 | 급등세 | 과열 주의, 추격 매수 신중 |
| +1% ~ +3% | 완만한 상승 | 상승 추세 확인, 매수 검토 가능 |
| -1% ~ +1% | 보합세 | 관망, 방향성 확인 필요 |
| -3% ~ -1% | 완만한 하락 | 저점 매수 기회 탐색 |

- 2025년 +12.2%: 본격적인 상승 추세 전환, 거래량도 동반 증가

> **인사이트:** 강남구 아파트 시장은 2023년 하반기를 저점으로 회복세에 진입했으며, 2024-2025년에 걸쳐 연 10% 이상의 상승률을 기록하고 있습니다. 다만 2025년 4분기에 상승폭이 둔화(+2.7%)되는 양상을 보이고 있어, 향후 추이를 주시할 필요가 있습니다.

### ㎡당 가격 비교의 중요성

아파트 단지를 비교할 때 총 거래가격만 보면 면적 차이에 의한 왜곡이 발생합니다. ㎡당 가격(또는 평당 가격)으로 환산하면 면적이 다른 단지도 공정하게 비교할 수 있습니다.

**환산 공식:**

```
㎡당 가격 = 거래금액 / 전용면적
평당 가격 = ㎡당 가격 x 3.3058
```

> **팁:** Claude Code에 "평당 가격으로 환산해서 비교해줘"라고 요청하면 자동으로 계산해 줍니다.

## 실전 응용

### 응용 1: 특정 면적대 필터링

전용면적 84㎡(구 34평형)는 국민평형으로 불리며, 거래가 가장 활발한 면적대입니다. 이 면적만 추출하여 분석하면 시장 추세를 보다 정확하게 파악할 수 있습니다.

**Claude Code 프롬프트:**

```
강남구 전체에서 전용면적 84㎡ (80~85㎡ 범위) 아파트의
최근 1년간 실거래가를 조회하고, 월별 평균가격 추이를 분석해줘.
상위 거래 단지 TOP 10도 함께 보여줘.
```

**기대 결과:**

- 강남구 84㎡ 타입 월별 평균가격 추이 테이블
- 거래가 활발한 단지 순위 (거래 건수 기준)
- 단지별 84㎡ 평균 거래가격 비교

이 분석을 통해 같은 면적대에서 어떤 단지가 상대적으로 저평가/고평가 되어 있는지 파악할 수 있습니다.

---

### 응용 2: 계절성 분석

부동산 시장에는 뚜렷한 계절성이 존재합니다. 일반적으로 봄(3-5월)에 거래가 활발하고, 여름(7-8월)과 겨울(12-1월)에는 거래가 줄어드는 패턴을 보입니다.

**Claude Code 프롬프트:**

```

어떤 달에 거래가 가장 활발하고, 어떤 달에 가장 적은지
계절성 분석을 해줘. 가격과 거래량의 상관관계도 같이 분석해줘.
```

**기대 결과:**


### 응용 3: 매매-전세 갭 분석

매매가와 전세가의 차이(갭)를 분석하면 투자 기회를 탐색할 수 있습니다. 갭이 적은 단지는 소액 투자가 가능하고, 갭이 큰 단지는 실거주 선호도가 높다는 의미입니다.

**Claude Code 프롬프트:**

```
대치동 은마아파트 76㎡ 타입의 최근 매매가와 전세가를 조회하고,
매매-전세 갭과 전세가율을 계산해줘.
최근 2년간 전세가율 추이도 함께 보여줘.
```

**기대 결과:**

```
[은마아파트 76.79㎡ 매매-전세 갭 분석]

현재 시세 (2026년 1월 기준)
─────────────────────────────────────────────────
 항목              | 금액
─────────────────────────────────────────────────
 최근 매매가       | 277,500만원 (27.75억)

 매매-전세 갭      | 125,500만원 (12.55억)
 전세가율          | 54.8%
─────────────────────────────────────────────────

전세가율 추이 (분기별)
─────────────────────────────────────────────────
 분기     | 매매 평균(만원) | 전세 평균(만원) | 전세가율
─────────────────────────────────────────────────
 2024 Q1  | 230,000        | 135,000        | 58.7%
 2024 Q2  | 238,000        | 138,000        | 58.0%
 2024 Q3  | 245,000        | 142,000        | 58.0%
 2024 Q4  | 252,000        | 145,000        | 57.5%
 2025 Q1  | 250,000        | 143,000        | 57.2%
 2025 Q2  | 258,000        | 145,000        | 56.2%

분석 요약:
- 전세가율이 58.7%에서 54.0%로 하락 추세입니다.
- 이는 매매가 상승 속도가 전세가 상승 속도보다 빠르다는 의미입니다.
- 갭이 확대되고 있어, 갭투자 관점에서는 투자 금액이 증가하는 상황입니다.
- 다만 전세가율 50% 이하로 하락하면 역전세 리스크에 유의해야 합니다.
```

**갭투자 핵심 개념:**

| 지표 | 계산 방법 | 투자 시사점 |
|------|----------|------------|
| 전세가율 | 전세가 / 매매가 x 100 | 70% 이상이면 갭투자에 유리 |
| 매매-전세 갭 | 매매가 - 전세가 | 실제 투자에 필요한 자기자본 |
| 갭 변동 추이 | 월별 갭 차이 | 갭 축소 시 투자 기회 |

> **주의:** 갭투자는 전세 보증금 반환 리스크, 금리 변동 리스크 등 다양한 위험 요소가 있으므로, 반드시 전문가 상담 후 신중하게 판단해야 합니다.

자주 사용하는 조회/분석 프롬프트를 정리합니다. Claude Code에서 그대로 복사하여 사용할 수 있습니다.

### 기본 조회

```
# 특정 단지 실거래가 조회
{지역}의 {아파트명} 최근 {기간}간 실거래가를 조회해줘

# 예시
강남구 대치동 은마아파트의 최근 1년간 실거래가를 조회해줘
```

### 비교 분석

```
# 단지 간 비교
{아파트1}, {아파트2}, {아파트3}의 시세를 비교 분석해줘

# 예시
대치동 은마아파트, 개포동 주공1단지, 잠실 엘스의 84㎡ 시세를 비교해줘
```

### 추세 분석

```
# 시계열 분석
{지역} {아파트명}의 최근 {기간}간 시세 추세를 분석하고,
이동평균선과 변동률을 계산해줘

# 예시
송파구 잠실동 잠실엘스의 최근 2년간 시세 추세를 분석하고,
3개월 이동평균선과 변동률을 계산해줘
```

### 경제지표 연동

```
# 금리와 시세 상관관계
한국은행 기준금리 변동과 {지역} 아파트 시세 추이를 함께 분석해줘

# 예시
최근 3년간 한국은행 기준금리 변동과 강남구 아파트 시세 추이를
함께 분석하고, 상관관계를 보여줘
```

## 주의사항

### 데이터 관련

- **신고 시차:** 실거래가는 계약일로부터 30일 이내에 신고하도록 되어 있으므로, 최근 1-2개월 데이터는 아직 반영되지 않은 거래가 있을 수 있습니다.
- **정정 신고:** 신고 후 가격이 정정되는 경우가 있으므로, 과거 데이터가 소급 수정될 수 있습니다.
- **해제 건수:** 계약 후 해제(취소)된 거래도 조회될 수 있으니, 해제 여부를 확인해야 합니다.
- **특수 거래:** 특수관계인 거래(친족 간 거래), 법인 거래 등은 일반적인 시세와 차이가 클 수 있습니다.

### API 관련

- **일일 호출 제한:** 국토교통부 API는 기본 일일 1,000건의 트래픽 제한이 있습니다. 대량 조회 시 호출 횟수를 관리해야 합니다.
- **월 단위 조회:** API는 월 단위로 데이터를 제공하므로, 1년치 데이터 조회 시 12번의 API 호출이 필요합니다.
- **서버 점검:** 공공데이터포털은 간헐적으로 서버 점검이 있으며, 이 경우 일시적으로 조회가 불가능할 수 있습니다.

### 투자 관련

- **면책 사항:** 이 튜터리얼에서 제공하는 분석 결과는 학습 목적이며, 실제 투자 판단의 근거로 사용해서는 안 됩니다.
- **전문가 상담:** 실제 부동산 투자 시에는 반드시 공인중개사, 세무사, 법무사 등 전문가와 상담하시기 바랍니다.
- **복합 요인:** 부동산 가격은 정부 정책, 금리, 수급, 개발 호재 등 다양한 요인에 의해 결정되므로, 실거래가 추세만으로 미래 가격을 예측하기는 어렵습니다.
- **세금 고려:** 취득세, 보유세(재산세/종합부동산세), 양도소득세 등 세금 부담을 반드시 사전에 확인해야 합니다.

## 문제 해결 (Troubleshooting)

### MCP 서버 연결 실패

```
오류: MCP 서버 'kr-realestate'에 연결할 수 없습니다.
```

**해결 방법:**
1. `.claude/settings.json`의 경로가 올바른지 확인합니다.
2. Python 환경에 mcp-kr-realestate 패키지가 설치되어 있는지 확인합니다.
3. `python mcp-servers/kr-realestate/main.py`를 직접 실행하여 오류 메시지를 확인합니다.

### API 키 인증 실패

```
오류: SERVICE_KEY_IS_NOT_REGISTERED_ERROR
```

**해결 방법:**
1. 환경변수 `MOLIT_API_KEY`가 올바르게 설정되어 있는지 확인합니다.
2. 공공데이터포털에서 API 활용 신청이 승인되었는지 확인합니다.
3. 인코딩 키가 아닌 **디코딩 키**를 사용하고 있는지 확인합니다.

### 데이터 조회 결과 없음

```
조회 결과가 없습니다.
```

**해결 방법:**
1. 지역코드가 올바른지 확인합니다 (시군구 5자리 코드).
2. 조회 기간에 해당 지역의 거래가 있었는지 확인합니다.
3. 아파트 단지명이 정확한지 확인합니다 (공식 등록 명칭 사용).
