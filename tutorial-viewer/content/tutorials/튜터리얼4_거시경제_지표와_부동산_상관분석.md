# 튜터리얼 4: 거시경제 지표와 부동산 상관분석

## 개요

- **목적:** 금리, GDP, 물가 등 거시경제 지표와 부동산 가격의 상관관계를 분석하여 투자 타이밍을 판단합니다.
- **대상 독자:** 거시경제 관점의 부동산 분석에 관심 있는 투자자, 데이터 기반 의사결정을 원하는 분
- **소요 시간:** 약 50-60분
- **난이도:** 중급-고급
- **사용 플러그인:**
  - `mcp-alphavantage` - 미국 경제지표(GDP, 금리, CPI, 실업률) 및 글로벌 금융 데이터
  - `mcp-kr-realestate` - 한국 부동산 실거래 데이터
  - `data:statistical-analysis` - 통계 분석(상관분석, 회귀분석)

## 사전 준비

### 1. Alpha Vantage API 키 발급

Alpha Vantage는 글로벌 경제지표와 금융 데이터를 제공하는 API 서비스입니다.

1. [alphavantage.co](https://www.alphavantage.co/) 에 접속하여 무료 계정을 생성합니다.
2. **Get your free API Key** 버튼을 클릭하여 API 키를 발급받습니다.
3. 발급받은 API 키를 환경변수로 설정합니다:

```bash
export ALPHAVANTAGE_API_KEY="your_api_key_here"
```

> **무료 플랜 제한사항:** 하루 25건, 분당 5건의 요청 제한이 있습니다.
> 본 튜터리얼은 약 15-20건의 API 호출을 사용하므로 무료 플랜으로 충분하지만,
> 반복 분석이 필요하다면 Premium 플랜(월 $49.99, 하루 150건)을 권장합니다.

`.claude.json` 또는 `claude_desktop_config.json` 파일에 다음 설정을 추가합니다:

## 핵심 배경 지식

### 부동산에 영향을 미치는 거시경제 지표

부동산 가격은 다양한 거시경제 변수의 영향을 받습니다. 각 지표의 작용 메커니즘을 이해하면 시장 움직임을 보다 체계적으로 분석할 수 있습니다.

#### 1. 기준금리 (가장 직접적 영향)

```
금리 인상 → 대출이자 증가 → 주택담보대출 부담 증가 → 매수 수요 감소 → 가격 하락 압력
금리 인하 → 대출이자 감소 → 주택담보대출 부담 감소 → 매수 수요 증가 → 가격 상승 압력
```

- 한국은행 기준금리는 부동산 시장에 **가장 강력한 영향**을 미칩니다.
- 금리 변동 후 부동산 가격에 반영되기까지 보통 **6~12개월의 시차**가 존재합니다.

**최근 한국 기준금리 추이:**

| 시점 | 기준금리 | 비고 |
|------|---------|------|
| 2021년 8월 | 0.75% | 코로나 이후 첫 인상 |
| 2022년 1월 | 1.25% | 연속 인상 |
| 2022년 7월 | 2.25% | 빅스텝(0.50%p 인상) |
| 2023년 1월 | 3.50% | 최고점 도달 |
| 2024년 10월 | 3.25% | 인하 전환 |
| 2025년 2월 | 2.75% | 연속 인하 |

#### 2. GDP 성장률

```
경제 성장 → 기업 이익 증가 → 고용 확대 → 가계 소득 증가 → 부동산 수요 증가
경제 침체 → 기업 이익 감소 → 고용 축소 → 가계 소득 감소 → 부동산 수요 감소
```

- GDP 성장률은 **중장기 부동산 수요**를 결정짓는 근본적 요인입니다.
- 한국 GDP 성장률: 2022년 2.6%, 2023년 1.4%, 2024년 2.0% (추정)

#### 3. 소비자물가지수 (CPI)

```
물가 상승(인플레이션) → 화폐 가치 하락 → 실물자산(부동산) 선호 → 가격 상승
물가 안정 → 실물자산 프리미엄 감소 → 가격 안정
```

- 인플레이션 헤지(hedge) 수단으로서 부동산의 가치가 부각됩니다.
- 다만 **과도한 인플레이션**은 금리 인상을 유발하여 오히려 부동산에 부정적입니다.

#### 4. 통화량 (M2, 광의통화)

```
통화량 증가(유동성 확대) → 시중 자금 증가 → 자산시장 유입 → 부동산 가격 상승
통화량 감소(유동성 축소) → 시중 자금 감소 → 자산시장 이탈 → 부동산 가격 하락
```

- 2020~2021년 코로나 대응 유동성 공급이 부동산 가격 급등의 핵심 원인이었습니다.
- M2 증가율과 부동산 가격 사이에는 약 **3~6개월의 시차**가 존재합니다.

#### 5. 실업률

```
실업률 하락 → 고용 안정 → 주택 구매 여력 증가 → 수요 증가
실업률 상승 → 고용 불안 → 주택 구매 위축 → 수요 감소
```

- 실업률은 부동산 수요의 **현실적 구매력**을 반영하는 지표입니다.

## Step 3: 상관계수 분석

### 상관계수란?

통계 분석에 앞서 핵심 개념을 이해합시다.

**피어슨 상관계수(r)**는 두 변수 사이의 선형 관계 강도를 -1에서 +1 사이의 값으로 나타냅니다.

| 상관계수 범위 | 해석 |
|-------------|------|
| +0.7 ~ +1.0 | 강한 양의 상관 (함께 상승) |
| +0.4 ~ +0.7 | 중간 양의 상관 |
| +0.1 ~ +0.4 | 약한 양의 상관 |
| -0.1 ~ +0.1 | 거의 상관 없음 |
| -0.4 ~ -0.1 | 약한 음의 상관 |
| -0.7 ~ -0.4 | 중간 음의 상관 |
| -1.0 ~ -0.7 | 강한 음의 상관 (반대로 움직임) |

### 기준금리 vs 아파트 가격 상관분석

**프롬프트:**
```
data:statistical-analysis 기능을 사용해서 한국은행 기준금리와 서울 아파트 평균 매매가의
피어슨 상관계수를 계산해줘. 동시점 상관계수와 시차 상관계수(lag 1~4분기)를 모두 구해줘.
```

**예상 결과:**

```
[기준금리 vs 서울 아파트 매매가 상관분석]

1. 동시점 상관계수: r = -0.42 (p-value: 0.058)
   → 중간 정도의 음의 상관. 금리가 오르면 부동산 가격이 하락하는 경향.
   → 다만 p-value가 0.05보다 커서 통계적 유의성은 경계선 수준.

2. 시차 상관분석 (기준금리 변동 → 아파트 가격 반응까지의 시차)
   ┌────────────────┬────────────┬──────────┐
   │ 시차(Lag)       │ 상관계수(r) │ p-value  │
   ├────────────────┼────────────┼──────────┤
   │ Lag 0 (동시점)  │ -0.42      │ 0.058    │
   │ Lag 1 (1분기)   │ -0.55      │ 0.012    │
   │ Lag 2 (2분기)   │ -0.68      │ 0.002    │ ← 최대 상관

   → Lag 2(6개월 시차)에서 상관계수가 -0.68로 가장 강합니다.

```

**해석:**

- **r = -0.68 (6개월 시차):** 기준금리가 올라간 뒤 약 6개월 후 부동산 가격이 하락하는 패턴이 통계적으로 유의미합니다.
- **p-value = 0.002:** 이 결과가 우연에 의한 것일 확률이 0.2%에 불과하므로 **통계적으로 매우 유의**합니다 (p < 0.01).

## Step 4: 회귀분석

### 다중 회귀분석이란?

회귀분석은 여러 독립변수(거시경제 지표)가 종속변수(부동산 가격)에 어떤 영향을 미치는지 수학적 모델로 설명하는 통계 기법입니다.

```
아파트 가격 = b0 + b1(기준금리) + b2(GDP) + b3(CPI) + b4(M2) + b5(실업률) + 오차
```

각 계수(b1~b5)가 양수이면 해당 지표 증가 시 가격도 상승, 음수이면 가격 하락을 의미합니다.

### 회귀분석 수행

**프롬프트:**
```
data:statistical-analysis를 사용해서 서울 아파트 평균 매매가를 종속변수로,
거시경제 지표 5개(기준금리, GDP 성장률, CPI, M2 증가율, 실업률)를 독립변수로 하는
다중 회귀분석을 수행해줘.

각 독립변수에는 앞서 확인한 최적 시차를 적용해:
- 기준금리: 2분기 전
- GDP 성장률: 1분기 전
- CPI: 3분기 전
- M2 증가율: 1분기 전
- 실업률: 2분기 전

모델 적합도(R², Adjusted R²), 각 변수의 계수와 p-value, VIF(분산팽창계수)를 보여줘.
```

**예상 결과:**

```
[다중 회귀분석 결과]

모델 적합도:
  R²           = 0.78  (독립변수들이 아파트 가격 변동의 78%를 설명)
  Adjusted R²  = 0.71  (표본 크기를 고려한 보정값)
  F-statistic  = 10.52 (p-value: 0.0003)

┌──────────────────┬─────────┬──────────┬──────────┬────────┐
│ 변수              │ 계수(β)  │ 표준오차  │ p-value  │ VIF    │
├──────────────────┼─────────┼──────────┼──────────┼────────┤
│ 절편(Intercept)   │ 10.25   │ 1.42     │ < 0.001  │  -     │
│ 기준금리(Lag 2)   │ -0.82   │ 0.24     │  0.003   │ 2.85   │
│ GDP 성장률(Lag 1) │ +0.35   │ 0.18     │  0.065   │ 1.52   │
│ CPI 상승률(Lag 3) │ +0.12   │ 0.15     │  0.438   │ 2.41   │
│ M2 증가율(Lag 1)  │ +0.58   │ 0.20     │  0.008   │ 1.78   │
│ 실업률(Lag 2)     │ -0.41   │ 0.22     │  0.075   │ 1.65   │
└──────────────────┴─────────┴──────────┴──────────┴────────┘

잔차 분석:
  Durbin-Watson  = 1.85  (자기상관 없음, 2에 가까울수록 양호)
  Shapiro-Wilk   = 0.96  (잔차 정규성 충족)
```

**결과 해석:**

1. **R² = 0.78:** 5개 거시경제 지표가 서울 아파트 가격 변동의 약 78%를 설명합니다. 나머지 22%는 정부 규제, 공급 물량, 심리적 요인 등 모델에 포함되지 않은 변수에 의한 것입니다.

2. **통계적으로 유의한 변수 (p < 0.05):**
   - **기준금리 (β = -0.82, p = 0.003):** 기준금리가 1%p 인상되면, 6개월 후 아파트 가격이 약 0.82억원(8,200만원) 하락합니다.
   - **M2 증가율 (β = +0.58, p = 0.008):** M2 증가율이 1%p 오르면, 3개월 후 아파트 가격이 약 0.58억원(5,800만원) 상승합니다.

3. **통계적으로 유의하지 않은 변수 (p > 0.05):**
   - **CPI (p = 0.438):** 소비자물가지수는 이 모델에서 독립적인 설명력이 부족합니다. 이는 CPI와 기준금리 사이의 **다중공선성(VIF=2.41)** 때문일 수 있습니다.

4. **VIF(분산팽창계수):** 모든 변수의 VIF가 5 미만으로, 심각한 다중공선성 문제는 없습니다. 다만 기준금리(VIF=2.85)와 CPI(VIF=2.41)의 상관이 다소 높으므로, CPI를 제거한 축소 모델도 고려할 만합니다.

### 이중축 차트 생성

**프롬프트:**
```
data:statistical-analysis를 사용해서 다음 차트를 생성해줘:
```

## 분석 시 주의사항

### 1. 상관관계와 인과관계의 구분

**상관관계(correlation)**는 두 변수가 함께 움직이는 경향을 나타낼 뿐, 하나가 다른 하나를 **원인(cause)**으로 유발한다는 것을 의미하지 않습니다.

```
예시: 아이스크림 판매량과 익사 사고는 높은 양의 상관을 보이지만,
     아이스크림이 익사를 유발하는 것이 아닙니다.
     둘 다 "여름 기온"이라는 공통 원인에 의해 증가합니다.

마찬가지로:
  M2 증가율 ↑ → 부동산 가격 ↑ 라는 상관이 있더라도,
  단순히 유동성만이 가격을 결정하는 것은 아닙니다.
  정부 정책, 공급 물량, 투자 심리 등 다양한 요인이 복합적으로 작용합니다.
```

### 2. 과거 패턴의 한계

- 역사적 상관관계가 미래에도 동일하게 유지된다는 보장은 없습니다.
- 특히 **구조적 변화**(structural break)가 발생하면 기존 패턴이 무효화될 수 있습니다.
- 예: 코로나19 팬데믹은 기존의 거시경제-부동산 관계를 크게 왜곡시켰습니다.

### 3. 비경제적 요인의 중요성

회귀분석에서 R²=0.76이라는 것은 **24%의 변동**이 모델에 포함되지 않은 요인에 의한 것을 의미합니다. 대표적 비경제적 요인:

| 요인 | 영향 |
|------|------|
| 대출 규제 (LTV/DTI/DSR) | 실수요자 구매력 직접 제한 |
| 세금 정책 (양도세, 종부세) | 매도/보유 의사결정에 영향 |
| 공급 정책 (재건축, 신도시) | 중장기 수급 밸런스 변화 |
| 인구 구조 (고령화, 1인가구) | 장기 수요 구조 변화 |
| 투자 심리/뉴스 | 단기 수요 변동 |

### 4. 분석 프레임워크 선택

```
단기 투자 (1년 미만):  금리, M2, 매수심리지수 중심
중기 투자 (1~3년):    금리 사이클, GDP, 공급 물량 중심
장기 투자 (3년 이상):  인구 구조, 도시화율, 소득 수준 중심
```

- 한국 경제 데이터는 발표 시점에 따라 **잠정치**와 **확정치**가 다를 수 있습니다.
- 부동산 실거래가 데이터는 신고 지연(최대 30일)으로 최신 데이터에 누락이 있을 수 있습니다.

## 부록: 용어 사전

| 용어 | 설명 |
|------|------|
| 피어슨 상관계수 | 두 변수 간 선형 관계의 강도와 방향을 나타내는 지표 (-1 ~ +1) |
| 시차 상관분석 (Lag Correlation) | 한 변수의 과거 값과 다른 변수의 현재 값 사이의 상관관계 분석 |
| 다중 회귀분석 | 여러 독립변수가 하나의 종속변수에 미치는 영향을 분석하는 통계 기법 |
| R-squared (R²) | 회귀모델이 종속변수의 변동을 설명하는 비율 (0~1, 높을수록 좋음) |
| p-value | 귀무가설(효과 없음)이 참일 때 관측된 결과가 나올 확률. 0.05 미만이면 통계적으로 유의 |
| VIF (분산팽창계수) | 다중공선성 정도를 나타내는 지표. 5 이상이면 다중공선성 문제 의심 |
| 다중공선성 | 독립변수끼리 높은 상관관계를 보여 회귀 계수의 신뢰성이 떨어지는 현상 |
| M2 (광의통화) | 현금 + 요구불예금 + 저축성예금 등을 포함한 넓은 범위의 통화량 |
| GDP 갭 | 실질 GDP와 잠재 GDP의 차이. 양수면 경기 과열, 음수면 경기 침체 |
| 선행지표 | 경제 변동에 앞서 변화하여 미래 경기를 예측하는 데 활용되는 지표 |
| DSR (총부채원리금상환비율) | 연소득 대비 모든 대출의 연간 원리금 상환액 비율 |
| 매수우위지수 | 부동산 시장에서 매수 우위 여부를 나타내는 심리 지표 (100 이상이면 매수 우위) |
